# painted prose — starter site

This is a hand-coded starter for your illustrated diary website.

## Pages
- `index.html` — Homepage with hero + tiles
- `journal.html` — Blog-style entries
- `portfolio.html` — Gallery grid
- `about.html` — Bio + statement
- `contact.html` — Contact form (Netlify-ready attributes if you ever host there)

## How to publish FREE on GitHub Pages

1) Create a free GitHub account at https://github.com
2) Click **New repository** → Name it **painted-prose** (or any name).
3) Upload all these files to that repository (drag & drop works).
4) Go to **Settings → Pages** →
   - Source: **Deploy from a branch**
   - Branch: **main** (or master) and **/root**
   - Save.
5) Wait a minute; GitHub Pages will give you a site URL like:
   `https://<your-username>.github.io/painted-prose/`

You can customize everything in `styles.css` and the HTML files to match your exact aesthetic. Add your images to `assets/images` and textures to `assets/textures`.
