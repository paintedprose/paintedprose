# Pink Theme Variant
This version matches your banner's pink palette.

## Image setup
- Put your banner image at `assets/images/banner.png` (use the wide header you shared).
- Put your wax-seal at `assets/images/seal.png` (transparent PNG works best).

## Favicon
The seal is referenced as the tab icon via `<link rel="icon" href="assets/images/seal.png">`.
Replace the file to update the favicon.

## Apply
Upload these files to your repo root (replace existing). Ensure images are placed as above.
